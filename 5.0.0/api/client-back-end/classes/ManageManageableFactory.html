getScheduleOccupiedTimePerDay

Defined in app/client/admin/js/ovManage/ManageableFactory.js:93

Syntax

getScheduleOccupiedTimePerDay

(
  • schedule
)
private

Summary

Gets all ranges of time a schedule occupies for each day of the week.

A schedule may start on a day and terminates another one or it may also start on a day and terminates the same day one week later.

Parameters:

  • schedule Object

    Schedule object with:

    • beginDate Date

      The begin date of the schedule

    • duration Number

      The schedule duration (in ms)

    • [recurrent] String optional

      Either 'daily' or 'weekly' if recurrent

    • [endDate] Date optional

      The end date of the daily schedule (required if recurrent is specified)
