ScheduleManager

Constructor

new ScheduleManager()

Manages process dependent scheduled jobs, for session recording, using node-shedule (https://www.npmjs.com/package/node-schedule).

Recording a job is process dependent, meaning that jobs have to be recreated if process is restarted.
