Summary

Adds an historic to device's history.

Use modifier addToGroup to also add the historic into history of device's group. Saves the history in database and cache then informs browsers about the new historic.

    The device

  • message String

    The historic message

    Message parameters for message translation

    true to also add the historic to the history of the device's group

start

Defined in app\server\Manager.js:1195

Syntax

start

(
  • callback
)
async

Summary

Starts manager.

When started the manager starts receiving events from both devices and browsers. It also populates cache with all registered devices and groups.

All expired schedules' jobs of devices and groups are removed. Other schedules are registered.

Parameters:

  • callback Function

    Function to call when it's done with :

    • Error An error if something went wrong, null otherwise
