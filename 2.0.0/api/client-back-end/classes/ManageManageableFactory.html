Summary

Checks if two schedules are in conflict.

Conflict examples :

schedule1 : [------------] schedule2 : [------------]

schedule1 : [------------] schedule2 : [------------]

schedule1 : [------------] schedule2 : [------------]

schedule1 : [------------] schedule2 : [------------]

schedule1 : [------------] schedule2 : [------------]

schedule1 : [------------] schedule2 : [--------]

schedule1 : [--------] schedule2 : [------------]

Parameters:

  • schedule1 Object

    Schedule object with :

    • beginDate Date

      The begin date of the schedule

    • duration Number

      The schedule duration (in ms)

    • [recurrent=false] Boolean optional

      true if this is a daily schedule, false otherwise

    • [endDate] Date optional

      The end date of the daily schedule (required if recurrent is set to true)

  • schedule2 Object

    Schedule object with :

    • beginDate Date

      The begin date of the schedule

    • duration Number

      The schedule duration

    • [recurrent=false] Boolean optional

      true if this is a daily schedule, false otherwise

    • [endDate] Date optional

      The end date of the daily schedule (required if recurrent is set to true)

Returns:

Boolean:

true if there are in conflict, false otherwise
