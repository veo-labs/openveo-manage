Installation

From OpenVeo root:

npm install @openveo/manage

Start / Restart OpenVeo

OpenVeo Manage is now installed. You can start / restart OpenVeo:

node server.js

Log to the back end

You can now log to the back end to access Manage plugin administration pages.
